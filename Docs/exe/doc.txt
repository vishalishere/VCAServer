运行服务器：点击直接运行即可, TCP 端口号9001

测试客户端:
	1.安装vc_redist.x86 (visual studio 2015) 
	2. 根目录下_Config/config.xml文件， 修改相机rtsp地址和服务器ip地址， 端口号不用改 
	3.点击运行


数据传输格式
|------------frame length -----------| 4 byte
|-------------- data  -------------- | 
|------------ frame length --------- | 4 byte
|-------------- data  -------------- | 

frame length 用大端格式表示, 例如: 255,0,0,0 = 255
			           0,1,0,0 = 256
			           1,1,0,0 = 257
